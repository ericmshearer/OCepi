# Apollo Label

Labels to match the style in theme_apollo.

## Usage

``` r
apollo_label(..., size = 4.5, color = "#231f20")
```

## Arguments

- ...:

  Other arguments available from geom_text.

- size:

  Font size, preset to 4.5

- color:

  Font color, preset to \#231f20

## Value

Standard label for plots.

## Details

Recommended settings: bar plot vjust = -0.3 horizontal bar plot hjust =
-0.5.
